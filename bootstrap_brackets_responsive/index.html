Guía para principiantes

Cómo prepararse para la aventura de Jin

Lecciones

Lección 1: Deja que la isla te guíe

Usa el viento y las pistas del entorno para orientarte en Tsushima.
Sigue a animales como aves doradas y zorros para encontrar sitios de interés.
Explora edificios escalables y utiliza el Viaje rápido para regresar
a áreas conocidas. Recolecta materiales para mejorar tus armas y
armaduras. Elige y personaliza tu equipo según las necesidades de cada situación.

Lección 2: Sobrevivir a los primeros combates

Prepárate para un combate brutal
donde incluso los enemigos
comunes pueden ser letales. Usa bloqueo, esquiva y desvío
para contraatacar efectivamente.
Aprende y domina las posturas
de combate para contrarrestar
los distintos tipos de enemigos y
aumentar tus posibilidades de victoria.

Lección 3: Primeras mejoras y relatos sugeridos

Busca relatos dorados para
obtener mejoras significativas
y nuevas habilidades. Prioriza
mejorar tu capacidad de
desvío y atención médica al
principio para sobrevivir.
Recluta aliados y explora el
Templo Dorado para obtener
armas, armaduras y valiosa
información de los ciudadanos refugiados.

Lección 4: La evolución de Jin y los secretos de Tsushima

Explora para encontrar
recursos y santuarios, supera
desafíos de bambú para
mejorar, y utiliza habilidades de Jin
como saltar para
acceder a lugares difíciles.
Escala santuarios para
obtener recursos valiosos,
visita termas para aumentos
en salud, sigue a los zorros
hacia recompensas, y usa
puntos de técnica para
mejorar la exploración.

Consejos

Intenta trepar tan alto como puedas para tener un lindo
vistazo de la isla, y luego toca la flauta de Jin. Darse un
momento de descanso para una bonita canción puede tener
efectos inesperados.
Los arqueros enemigos siempre gritarán para alertar a sus
aliados antes de disparar una flecha. Escucha siempre con
atención para poder esquivar y quizás también disparar una
flecha.
Cuando termina una batalla, puedes usar el touch pad para
limpiar o enfundar tu espada o para presentarle tus respetos a
un enemigo caído.
